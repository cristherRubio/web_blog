Python Project - Clash Report.

The most interesting project so far and also the one I have dedicated the most time. The program creates an app with Tkinter that dynamically connects to the APS("Autodesk Platforming Services") API´s. It allows the user to connect to the company (construction) projects and request clashes between BIM("Building Model Information") models, and from this finally generates CSV files that allows the company to evaluate the contracters solution to this clashes across time.

Python Project - Procore PDF Upload.

The challenge in this project was to create a simple windows app that allows the user to bypass the PDF manual upload in procore website. This works by generating several requests to the Procore´s API and let the user select a folder from which to upload the PDF, then parsing the file names as string insert the code and name of the PDF drawing to the project folder. Saving a lot of time to the end-user.
